# chainlit-chatbot


## How to run ?

1. Create a virtual environment

```bash
conda create -n llmapp python==3.11 -y
```

2. Activate the virtual environment

```bash
conda activate llmapp
```

3. Install the requirements

```bash
pip install -r requirements.txt
```


## Chainlit command

```bash
chainlit init
```

```bash
chainlit run app.py
```
